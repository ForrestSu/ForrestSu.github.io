1 MarkdownEditing Markdown高亮显示

安装完后需修改GFM Settings-User文件，添加我们自己个性化的配置：

添加如下内容：

{ “color_scheme”: “Packages/MarkdownEditing/MarkdownEditor-Dark.tmTheme”, “draw_centered”: false, “line_numbers”: true, “highlight_line”: true, //wrap_width表示多少个字符显示到下一行,0表示不限制 “wrap_width”:0 }

最后我们新建一个md文件就可以开始写作了^_^

2 安装markdown-preview

这个插件可以把Markdown书写的文本输出为github样式html文件，作用：本地开发Git项目预览readme的效果。

git clone https://github.com/revolunet/sublimetext-markdown-preview MarkdownPreview

然后可以修改MarkdownPreview的settings-user文件,开启部分我们需要的功能。

{ “enable_mathjax”: true, “enable_highlight”: true }

最后，我们在Preferences-key bindings-user里面自定义一个快捷键，可以快速输出html。

//4 markdown_html预览 { “keys”: [“alt+c”], “command”: “markdown_preview”, “args”: {“target”: “browser”, “parser”:“markdown”} },

3 最后安装OmniMarkupPreviewer

添加自定义配置OmniMarkupPreviewer ->Setttings - User：

{
	"mathjax_enabled": true,
	"renderer_options-MarkdownRenderer": {   
       "extensions": ["strikeout","tables","extra","codehilite", "meta", "toc", "smarty"]
    }
}

这插件支持实时在线预览，同样自定义快捷键key bindings-user：

//5 markdown 在线预览 { “keys”: [“alt+x”], “command”: “omni_markup_preview”, “context”: [{“key”: “omnimarkup_is_enabled”, “operator”: “equal”, “operand”: “"}] }
