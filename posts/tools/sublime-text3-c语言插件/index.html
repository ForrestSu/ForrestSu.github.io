0 sublime-text-git安装

ctrl+shift+p 输入pci，然后搜索git直接安装即可。 这个插件可以直接使用git命令，很方便，git log 显示所有commit的记录，继续输入还支持搜索哦（介个功能很不错）。

官方目前还没更新支持ST3，安装后在Preferences ->key bindings-User中加入如下一句，然后选中c++的代码，按ctrl+d 就可以按等号(=)对齐。 >//1.Alignment 代码自动对齐 { “keys”: [“alt+d”], “command”: “alignment” }

3 C Improved

更加丰富的C语言着色方案

然后点击右下角c++，可将*.cpp文件显示高亮都用cimproved

4 DocBlockr

自动生成函数注释，在不同参数之间切换可以用tab.

5 AllAutocomplete

AllAutocomplete可以对打开的所有文件的变量名进行提示。sublime默认只对当前打开的文件的变量进行提示。
