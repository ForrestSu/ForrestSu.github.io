1 UML图

我们先看用ST3书写脚本生成的UML图效果，写以下脚本保存为test.wsd文件 >@startuml
skinparam defaultFontName 宋体 用户 -> Bob: Authentication Request Bob -> Alice : Hello, how are you Consolas Alice -> Bob : Fine, thank you, and you? @enduml

2 安装步骤

2.1 下载[graphviz-2.38.zip

](http://www.graphviz.org/Download_windows.php)(`注：我用的是Windows,如果是其他操作系统选择对应版本`) 目前最新版是graphviz-2.38(2016年5月15日) 解压到系统的某一目录，然后添加到系统环境变量(如下图),重启OS后生效：

3 安装ST3插件sublime_diagram_plugin

通过Preferences->browse packages进入插件安装包，下载完成后重启ST3: >git clone https://github.com/jvantuyl/sublime_diagram_plugin.git

4 修改配置参数

在 Preferences ->key bindings-User中新增快捷键设置: >{ “keys”: [“alt+z”], “command”: “display_diagrams”}

还需根据具体的操作系统，进行图片查看器和编码的修改(下面截图以windows为例)，点击diagram->Settings-default：

安装完成。
